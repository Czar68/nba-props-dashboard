# Betting Dashboard

Production betting operations dashboard built with Next.js, TypeScript, and Tailwind CSS.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start development server:
```bash
npm run dev
```

3. Navigate to `http://localhost:3000`

## Features

- Real-time betting recommendations
- Advanced filtering by sport, structure, EV, Kelly, risk level
- Bankroll management with risk visualization
- Performance metrics and correlation filter summaries
- Auto-refresh every 5 minutes

## Data Source

Dashboard reads from `dist/dashboard_data.json` generated by the daily betting runner script.

## Build

```bash
npm run build
npm start
```

## Dependencies

The dashboard requires these dependencies to be installed:
- next
- react
- react-dom
- typescript
- tailwindcss
- lucide-react
- date-fns

Run `npm install` to install all required packages.
